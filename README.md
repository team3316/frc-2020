# Controls
## X-Box Controller
* RT - Warmup & Shoot when held
* LT - Shoot without warmup when held
* BACK - Bottom Port Shoot when held
* RB - Shoot with vision when held
* LB - Toggle Intake
* X - Rotate Pizza Clockwise when held
* A - Rotate Pizza Counter Clockwise when held
* B - Open Climber Arm
* Y - Climb when held

## Joysticks
* Right Trigger - Hold and move Right Joystick for Straight Drive
* Left Trigger - Toggle Gear
* Right Thumb Button - Toggle Auto Align with Vision (press again to cancel alignment)
